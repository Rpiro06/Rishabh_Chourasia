Sports Trial Assessment

Role: Front-end Developer • Focus: Interactive Evaluation Design

Challenge

Create a visually engaging sports trial assessment experience that works well across devices and clearly guides users through each step. The project needed a polished web page to present trial criteria, scoring, and next steps for participants.

Built a responsive interface using HTML and CSS, with refined typography, interactive hover states, and a clear progress section. Advanced front-end logic was expressed through CSS transitions and stateful layouts to support a seamless evaluation flow.
